1. We make a HTTP request to a website: exmaple -> youtube.com
2. We get back a resposne from the web server that youtube.com is associated with
3. The response is a web page (front-end)
4. A web page consists of: HTML, CSS and Javascript
5. HTML = Content of the page (text, buttons, form inputs, images, hyperlinks)
6. CSS = styling of the content (layout, colors, layout)
7. Javascript = Interactivity (clicking, hovering, scrolling, typing) between the 
user and web page which results in modification of the HTML and CSS
